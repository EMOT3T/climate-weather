
# Climate Weather

This code aims to create a simple weather page.

The main purpose of this project is to test APIs and direct user interactions with JS.


------

## Company API:
- Open Weathermap:
- https://openweathermap.org/

----

## API features
- Access to real-time weather data.
- Climate predictions for the future.
- Temperature, humidity, wind and cloud maps.
- Detailed information about the current weather.
- Historical climate data.
- Weather alerts.
- Custom weather maps.

-------

## How you can use it in your projects

Create and develop your projects using the same verification standards and data processing demonstrated here.

Have this project as a simple path to give you direction, but use it in your own way, for your project and to solve your problem.
